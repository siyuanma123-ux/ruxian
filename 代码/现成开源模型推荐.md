# 现成可用的开源乳腺癌诊断模型推荐

## ⭐ 推荐模型（按易用性排序）

### 1. MV-Swin-T (Multi-view Swin Transformer) ⭐⭐⭐⭐⭐
**最推荐 - 现代PyTorch实现，易于使用**

- **GitHub**: https://github.com/prithuls/MV-Swin-T
- **论文**: https://arxiv.org/abs/2402.16298
- **特点**:
  - ✅ 使用PyTorch，现代框架
  - ✅ 专门为乳腺X光片设计
  - ✅ 多视图信息融合
  - ✅ 在CBIS-DDSM和Vin-Dr Mammo上表现优异
- **准确率**: 在公开数据集上表现优异
- **安装难度**: ⭐⭐ (简单)
- **使用方式**:
  ```bash
  git clone https://github.com/prithuls/MV-Swin-T.git
  cd MV-Swin-T
  # 按照README安装依赖和下载权重
  ```

### 2. End-to-end All Convolutional Design ⭐⭐⭐
**高准确率但需要旧版本依赖**

- **GitHub**: https://github.com/lishen/end2end-all-conv
- **论文**: https://arxiv.org/abs/1708.09427
- **特点**:
  - ✅ 高准确率 (AUC 0.95-0.96)
  - ✅ 有预训练权重
  - ❌ 需要Keras 2.0.8和TensorFlow 1.x
  - ❌ 需要Python 3.7
- **准确率**: 
  - INbreast: AUC 0.95 (单模型), 0.96 (2模型平均)
  - DDSM: AUC 0.86-0.88 (单模型), 0.91 (3模型平均)
- **安装难度**: ⭐⭐⭐⭐ (较难，需要旧版本环境)
- **权重下载**: https://drive.google.com/drive/folders/0B1PVLadG_dCKV2pZem5MTjc1cHc
